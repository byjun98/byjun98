<details>
<summary><b>프론트엔드 기술 상세 (펼치기)</b></summary>

#### 카카오맵 — 뷰포트 시스템 설계

```
사용자 위치(center) → 검색 기준점 → Places API → 검색 결과(search[])
       │                                              │
       ▼                                              ▼
   현재 위치 마커                              결과 마커 렌더링
       │                                              │
       └──── Bounds 계산 ── setBounds() ──── 뷰포트 자동 조정
```

- `setBounds()` = 여러 오브젝트를 모두 포함하도록 뷰포트 조정
- `panTo()` = 특정 오브젝트로 카메라를 부드럽게 이동
- `lastCenter` 캐시 = 카메라 위치를 별도 변수에 캐시하여 뷰 전환 시에도 유지하는 패턴

#### AI 분석 → 챗봇 전환 파이프라인

```
[사용자] → 이미지 선택 → [Camera.js] → FormData POST → [Backend] → Python AI → 견종명 반환
```
