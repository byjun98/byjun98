### 1. [애니멀 케어 플러스] (2024.03 ~ 2024.06)

<a href="[프로젝트 Github 링크]">
  <img src="[프로젝트 대표 이미지 주소]" alt="프로젝트 1 이미지" width="700"/>
</a>

**주요 기능**

- **AI 종판별**: Python으로 강화학습한 YOLO모델로 업로드한 강아지사진의 종을 판별
- **종별 케어법 제공**: 판별된 강아지종을 챗봇에게 전달, 맞춤형 케어법 제공
- **AI 피부병 판별**: Python으로 강화학습한 YOLO모델로 업로드한 강아지사진의 피부병을 판별
- **상점 연동**: 쇼핑몰과 링크연결, 결제 API를 활용한 페이 결제 기능 제공
- **애견지도 제공**: 사용자 위치를 geolocation으로 받아와 주변 애견동반 카페, 식당과 동물병워 산책로등 핀으로 표시하여 제공
- **커뮤니티기능 제공**: 강아지의 사진을 올리며 소통할 수 있는 게시판 제공

**역할**

- react 개발
- JWT 기반의 사용자 인증 시스템 구현
- CRUD 기능이 포함된 게시판 기능 개발
- 카카오맵 api로 사용자 중심 지도 구현
- gpt api로 인공지능 챗봇 구현

**사용한 기술 스택**

- **Backend**: `Java`, `Spring Boot`
- **Frontend**: `React`
- **Database**: `MySQL`
- **AI**:`Python`
- **Deployment**: `AWS EC2`, `Docker`

**성과**

- JWT 도입을 통해 보안 수준을 향상시키고 세션 관리 비용 절감
- N+1 문제 해결 등 쿼리 최적화를 통해 API 응답 속도 50% 개선

  <img src="https://raw.githubusercontent.com/byjun98/foot_pressure_arduino/main/assets/3d.png" alt="프로젝트 2 대표 이미지" width="700"/>

**프로젝트 스크린샷**

| 2D 차트 (실시간)                                                                                  | 3D 차트                                                                                        | 녹화 중                                                                                             |
|:--------------------------------------------------------------------------------------------:|:--------------------------------------------------------------------------------------------:|:------------------------------------------------------------------------------------------------:|
| ![2D 차트](https://raw.githubusercontent.com/byjun98/foot_pressure_arduino/main/assets/2d.png) | ![3D 차트](https://raw.githubusercontent.com/byjun98/foot_pressure_arduino/main/assets/3d.png) | ![녹화](https://raw.githubusercontent.com/byjun98/foot_pressure_arduino/main/assets/recording.png) |

**주요 기능**

- **실시간 족압 시각화**: 4x4(16개) 압력 센서 데이터를 BLE로 수신해 2D/3D 차트로 실시간 시각화
- **데이터 녹화/분석**: 10초간 족압 데이터를 녹화하고 별도 화면에서 동일 차트로 재분석
- **센서 선택 분석**: 전체 센서 또는 단일 센서를 선택해 압력 변화 추이 확인
- **AI 건강 상담 챗봇**: OpenAI GPT 기반으로 족압/발 건강 관련 상담 기능 제공
- **건강 뉴스 연동**: 네이버 뉴스 API 연동으로 발 건강 관련 최신 정보 제공
- **하드웨어 없는 데모 모드**: 랜덤 데이터 생성기로 BLE 장비 없이도 앱 기능 검증 가능

**역할**

- React Native + Expo 기반 모바일 앱 아키텍처 설계 및 구현
- 아두이노/ESP32 BLE 데이터 파싱 및 상태관리(Zustand) 연동
- ECharts(WebView) 기반 2D/3D 실시간 차트 렌더링 구현
- 녹화 기능(타이머/스냅샷/화면 전환) 및 통계 UI(최소·평균·최대) 구현
- 챗봇(OpenAI API)·뉴스(네이버 API) 기능 통합 및 예외 상황 대응 로직 적용

**사용한 기술 스택**

- **Frontend**: `React Native`, `Expo`, `TypeScript`
- **State Management**: `Zustand`
- **Visualization**: `ECharts`, `ECharts-GL`, `WebView`
- **Communication**: `BLE (react-native-ble-plx)`
- **Animation/UI**: `Reanimated`, `Moti`, `NativeWind`
- **Hardware**: `Arduino`, `ESP32`, `4x4 Pressure Sensor Matrix`
- **API**: `OpenAI GPT`, `Naver Search API`

**성과**

- Context API → Zustand 전환으로 센서 데이터 갱신 시 불필요한 전역 리렌더링을 줄여 화면 반응성 개선
- 차트 로직 공통 컴포넌트화 및 TypeScript 전환으로 유지보수성/확장성 향상
- 녹화 시점 클로저 버그, 빈 데이터 렌더 루프, API 키 미설정 예외 등 핵심 안정성 이슈 해결

### 3. [GameMatch (ChuraiGame)] (2025.09 ~ 진행중)

<a href="https://github.com/byjun98/ChuraiGame">
  <img src="https://raw.githubusercontent.com/byjun98/ChuraiGame/main/erd.png" alt="GameMatch 대표 이미지" width="700"/>
</a>

**주요 기능**

- **Steam 연동 개인화 추천**: OpenID 기반 Steam 로그인 후 라이브러리/최근 플레이 데이터를 반영해 맞춤 추천 제공
- **온보딩 기반 취향 수집**: 왓챠 스타일 4점계(-1/0/3.5/5) 평가로 콜드스타트 유저 즉시 추천
- **하이브리드 추천 엔진**: 게임 간 협업필터링 + 장르/태그 + 메타크리틱 점수를 결합한 하이브리드 유사도 계산
- **AI 큐레이터 챗봇**: 사용자 컨텍스트를 활용한 GPT 기반 추천 대화 및 게임 설명 번역 API 제공
- **세일/최저가 정보 제공**: CheapShark 연동으로 할인 정보 조회 및 세일 섹션 제공
- **커뮤니티 기능**: 게시글/댓글/좋아요/이미지·동영상 업로드 기반 커뮤니티 운영

**역할**

- Django + DRF 기반 백엔드/API 설계 및 구현
- Vue(CDN) + Django Template 기반 메인 대시보드 UI/상태 흐름 구현
- Steam/Google/Naver OAuth 연동 및 계정 연결/해제 플로우 구현
- 추천 배치 파이프라인(희소행렬 + 코사인 유사도 + Top-K 저장) 구축
- RAWG 응답 캐시 레이어(CachedGameList) 구현으로 메인 로딩 최적화

**사용한 기술 스택**

- **Backend**: `Python`, `Django`, `Django REST Framework`
- **Frontend**: `Vue 3 (CDN)`, `Django Templates`, `Tailwind CSS`
- **Database**: `SQLite`
- **Data/ML**: `pandas`, `NumPy`, `SciPy`, `scikit-learn`
- **API 연동**: `Steam Web API(OpenID)`, `RAWG`, `CheapShark`, `OpenAI`, `Google Gemini`, `Google OAuth`, `Naver OAuth`

**성과**

- RAWG 호출 결과를 DB 캐싱하여 메인 목록 로딩 체감 속도 개선(README 기준 약 98% 단축)
- 온보딩 + Steam 연동 이중 경로로 콜드스타트 사용자도 즉시 추천 경험 제공
- 추천 유사도 테이블 정규화/랭킹 저장으로 조회 성능과 확장성 고려한 구조 확보

🔗 **[GitHub Repository]**(https://github.com/byjun98/ChuraiGame) | 🔗 **[배포 링크]**(개발/시연 환경 기준, 별도 운영 링크 준비 중)
